CharAnalyzer

Character, Letter, and Word Analyzer

The character analyzer is a 100% free tool that you can use to analyze your text. This tool will analyze each character in the text that you enter in the dialogue box and distinguish the type of each character. For each type of character will display the percentage of occurrence in your text.

To find the percentage of each character of your writing, simply copy and paste text into the tool or write directly into the text area. Once done, the free online tool will analyze your text and display the amount of each type character in percent.
